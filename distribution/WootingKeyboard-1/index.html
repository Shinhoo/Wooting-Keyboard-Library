WootingKeyboard

A Library by Shinhoo Park for the Processing programming environment.
Last update, 02/01/2018.

This library can be used to analyze the raw data of the Wooting Keyboard.
The library will trigger onReadEvents when recieve raw data from Wooting keyboard. This data can be parsed into data of type String or HashMap through parseLog and parseData functions.

Download

Download WootingKeyboard version 1.0.0 (1) in .zip format.

Installation

Unzip and put the extracted WootingKeyboard folder into the libraries folder of your Processing sketches. Reference and examples are included in the WootingKeyboard folder.
